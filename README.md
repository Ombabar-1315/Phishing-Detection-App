# 📱 Mobile Based Phishing Detection System

## 🔐 Project Overview

The **Mobile Based Phishing Detection System** is an Android application designed to protect users from phishing attacks by analyzing suspicious URLs before they are opened. The application detects malicious links received through SMS, clipboard, or manual input using various network security techniques.

This project combines **Mobile Application Development** with **Cyber Security concepts** to provide real-time protection against online threats.

---

## 🎯 Objectives

* To detect phishing URLs using security analysis techniques
* To alert users before accessing malicious websites
* To improve user awareness about cyber threats
* To provide a simple and user-friendly security solution

---

## 🚀 Features

* URL Input and Scanning
* SMS Link Detection
* Clipboard Monitoring
* Real-Time Phishing Detection
* HTTPS Security Verification
* Domain and Keyword Analysis
* Blacklist Checking (API-based)
* Risk Score Calculation
* Detailed Result Display (Safe / Suspicious / Phishing)
* Scan History Storage
* User Alert System
* Phishing Reporting Feature

---

## 🧠 Technologies Used

* **Frontend:** Android (Java / Kotlin, XML)
* **Backend (Optional):** Firebase / REST API
* **Security APIs:** Google Safe Browsing API, PhishTank
* **Database:** SQLite / Firebase
* **Tools:** Android Studio, Git, GitHub

---

## 🏗️ System Architecture

```
User Input (URL / SMS)
          ↓
Android Application
          ↓
URL Analyzer Module
          ↓
Security Engine
          ↓
API / Blacklist Check
          ↓
Risk Calculation
          ↓
Result Display
```

---

## 📱 Application Flow

1. User enters or scans a URL
2. App extracts domain and protocol
3. Security checks are applied
4. URL is verified using APIs
5. Risk score is calculated
6. Result is displayed with warning if needed

---

## 🔐 Security Techniques Used

* URL Parsing
* SSL Certificate Verification
* Domain Reputation Analysis
* Keyword-Based Detection
* Blacklist Checking
* Risk Score Algorithm

---

## 📊 Example Output

```
URL: http://login-bank-security.xyz

Status: PHISHING ⚠  
Risk Level: HIGH  

Reasons:
• Suspicious keywords detected
